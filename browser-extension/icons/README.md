# Extension Icons

This directory should contain the following icon files for the PhishAnalyzer extension:

- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)  
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Recommended Icon Design

Use a shield icon (🛡️) with the following color scheme:
- Primary color: #0078d4 (blue)
- Secondary color: #f3f2f1 (light gray)
- Accent color: #107c10 (green for safe) or #d13438 (red for danger)

## Temporary Solution

For immediate testing, you can use simple colored squares or download free security icons from:
- https://www.flaticon.com/
- https://feathericons.com/
- https://heroicons.com/

Make sure to rename the downloaded files to match the required names above.
